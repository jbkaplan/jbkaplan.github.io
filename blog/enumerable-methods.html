Enumerable#map

Posted by Jonathan Kaplan on March 13, 2016

The Enumerable#map method runs a block of code, an enumerator, and iterates that code block to each element. It returns a new array. This method can also be run using #collect, as it is the exact same method.

In action, it looks like this:

array = [5, 10, 15, 20]

array.map { |number| number + number}

Returns: #=> [10, 20, 30, 40]

What's really important with the #map method is that it changes the return value, and returns an array. This is also what makes it different from the #each method. #each gives you a little more flexibility overall, but if you are looking to change the return value of an array, #map is the way to go.
