What are the best practices associated with using classes vs. ids?

Posted by Jonathan Kaplan on February 28, 2016

To begin, the main differences between using classes and ids are that classes can be used on multiple elements to indicate a more global style. Ids are different in that you should only use an id for one element. That one element is unique, and you should not use that same id on another element, it is not best practice.

A few best practices associated with using classes vs. ids are:

  • Classes are NOT unique
    • If more than one element will require the same style, make it a class
  • Ids ARE unique
    • You know you are calling out one specific element
  • Elements can have BOTH classes and ids
  • Ids can be used to link, classes cannot
    • This can be useful to link to a certain part of a page (i.e. a different section)
